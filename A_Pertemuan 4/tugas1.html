Belajar Coding dengan HTML dan CSS

Dasar-dasar HTML

HTML (Hypertext Markup Language) adalah bahasa standar untuk membuat halaman web.

HTML terdiri dari elemen-elemen yang dinyatakan dengan tag.

Paragraf ini menggunakan Inline CSS untuk memberi warna latar belakang dan warna teks yang berbeda

Langkah-langkah Memulai Coding

  • Pasang text editor (misalnya, VSCode atau Sublime Text)
  • Buat file HTML pertama anda
  • Tambahkan elemen dasar seperti <h1>, <p>, dan <div>
